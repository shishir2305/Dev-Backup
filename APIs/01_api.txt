An Application Programming Interface (API) is a set of commands, functions, protocols and objects, that programmers can use to create software or interact with an external system

Endpoint - In simple terms, an API endpoint is the point of entry in a communication channel when two systems are interacting.  It refers to touchpoints of the communication between an API and a server. The endpoint can be viewed as the means from which the API can access the resources they need from a server to perform their task. An API endpoint is basically a fancy word for a URL of a server or service.
